Implementation Guide

Build, configure, and run MailFolio.Console. Includes troubleshooting and operational notes.

What MailFolio.Console does

  • Loads SMTP config from MAILFOLIO_KEYFILE (JSON).
  • Reads passphrase from MAILFOLIO_PASSPHRASE.
  • Decrypts UsernameEnc and PasswordEnc using KeyFolio.
  • Sends email via SMTP using MailKit.
  • On failure, writes a sanitized error record to SQLite (no body/subject/credentials stored).
Good to know: the SQLite “sent mail error tracking” DB is written only on failures. Successful sends do not create rows.

Where to start

  1. Go to Getting Started for build + first run.
  2. Go to Configuration for env vars and key file schema.
  3. Use Runbook for common run patterns.
  4. Use Troubleshooting for failure codes + diagnostics.
