Troubleshooting

Common issues you may hit while encrypting/decrypting envelope strings.

I can’t decrypt an envelope

Check these in order:

  1. Passphrase mismatch (most common)
  2. Different options/version between generator and validator
  3. Modified/truncated envelope (whitespace, line breaks, field limits)
Tip: Some systems silently truncate long strings. Re-copy and verify the full value exists.

I lost my passphrase

There is no recovery. Envelopes cannot be decrypted without the original passphrase.

It keeps prompting me

  • Env var not set: KEYFOLIO_SECRET
  • New app session: expected behavior

Copy Output didn’t work

  • Clipboard restrictions (remote sessions / locked-down environments)
  • Output is empty
