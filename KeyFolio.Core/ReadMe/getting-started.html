KeyFolio.Core Guide
Build and basic consumption patterns.

Getting Started

Build the library and consume it from an application project.

Build

dotnet clean
dotnet restore
dotnet build -c Release

Consumption pattern

KeyFolio.Core is meant to be referenced from apps (MailFolio.Console now; KeyFolio.Console later). The app is responsible for supplying the secret (passphrase) via ISecretProvider.

Minimal example (conceptual)

// Pseudocode: exact names may vary by project
var options = new KeyFolioOptions();
var crypto  = new KeyFolioCrypto(options);

// Secret provider supplies passphrase (env, prompt, DPAPI, etc.)
ISecretProvider provider = new EnvSecretProvider("MAILFOLIO_PASSPHRASE");

var plain = crypto.Decrypt(envelopeText, provider);
Important: Your apps should treat the passphrase as a secret: don’t write it to disk, don’t log it, and keep it in memory only as long as needed.
