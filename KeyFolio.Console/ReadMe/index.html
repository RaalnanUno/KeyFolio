KeyFolio.Console

Command-line implementation of the KeyFolio.Core crypto library.

What this is

KeyFolio.Console is an implementation of KeyFolio.Core. It provides a CLI for generating, validating, and inspecting encrypted envelope strings.

  • KeyFolio.Console → CLI implementation
